# 因果关系

+ because, since, for, given, given that, result from, in that, by virtue of + 原因
+ so, consequently, therefore, result in + 结果

# 递进关系

+ if not = even
  + 注意程度递进，递进的两边不能是相同程度的
  + 递进提示词：even, downright
+ not just ..., but ... 不仅而且
+ not A or B: 不是A也不是B

肯定递进与否定递进

+ 肯定递进：A → B
  + B 比 A 程度深
  + 她是班花，甚至是校花
+ 否定递进：A → B
  + A 比 B 程度深
  + 她不是校花，甚至不是班花
