角和直线的关系

+ 多边形内角和 = （边的数量/角的数量-2）$\times$ 180°

三角形外角和内角之间的关系：

+ 三角形的外角 = 三角形中两个不相邻的内角之和

直线平行与角的关系：

+ 同位角相等
+ 内错角相等
+ 同旁内角互补

三角形的性质：

+ 大边对大角，大角对大边
+ 两边之和大于第三边，两边之差小于第三遍
+ 勾股定理
  + 直角三角形：$a^{2} + b^{2} = c^{2}$
  + 锐角三角形：$a^{2} + b^{2} ＞ \ c^{2}$
  + 钝角三角形：$a^{2} + b^{2} ＜\  c^{2}$

三角形的面积：

+ (1/2)×底×高
+ (1/2)×a×b×sinC

## 面积公式

长方体：

+ 体积：$abc$
+ 表面积：$2\times(ab+bc+ac)$

正方体：

+ 体积：$a^{3}$
+ 表面积：$6a^{2}$

圆柱

+ 体积：$ \pi r^{2} h$
+ 表面积：$2 \pi r^{2} + 2 \pi rh$

球

+ 体积：$\frac{4}{3} \pi r^{3}$
+ 表面积：$4 \pi r^{2}$

圆锥

+ 体积：$\frac{1}{3} \pi r^{3}$
